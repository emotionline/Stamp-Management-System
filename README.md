# Stamp-Management-System

안녕하세요, 돌봄센터에서 근무하고 있는 emotionline입니다.

저희 돌봄센터에서는 매달마다 스티커 시장이 있습니다.

아이들에게는 몹시 즐거운 축제와 다름이 없습니다.
아이들은 자신이 평소에 사고 싶었던 레고를 손에 쥐기도 하고, 슬라임을 담기도 하고, 왁뿌볼이나 키캡을 찾습니다.

그런데, 이런 날을 온전하게 즐기지 못하는 아이들이 때때로 존재합니다.
사실, 꽤나 빈번하죠.
도장을 잃어버리는 까닭입니다.
저희 돌봄센터만 하더라도 벌써 10차례를 넘겼는데요.

거의 매달마다 한 아이는 잃어버립니다.
그러다 보니, 자신의 도장 개수를 기억하지 못하는 아이들은 억울한 일이 발생합니다.
선생님들도 그런 억울한 일이 없도록 노력하다가, 다른 아이들이 "왜 도장을 찍어주냐고" 말을 해서 억울한 상황이 연출되기도 하죠.

저 역시, 그런 사례를 겪다 보니. "아, 아이들이 직접 도장을 찍는 손맛은 지켜주면서도, 혹시 모를 분실을 방지할 수 있다면 얼마나 좋을까" 생각했습니다.

하지만, 수기로 매일 작성하는 것은 돌봄센터의 업무 상 꽤 부담이 많이 가기에 미뤄오고 있었는데.
2026년 6월 9일 화요일, 오늘 한 아이가 도장을 잃어버림을 통해서 "도장 관리 시스템"을 구축하게 되었습니다.

===

위 프로그램의 작동 방식은 단순합니다.

1. 회원가입 칸에서 아이디와 비밀번호를 설정합니다.
2. 로그인 칸에서 아이디와 비밀번호를 입력합니다.
3. "새로운 아이 추가" 기능을 활하여, 아이들을 "실시간 도장 현황판"에 추가합니다.
4. "간편 도장 수량 조절" 기능을 활용합니다.
5. 초기화를 해야 되는 때가 오면, "초기화" 버튼을 활성화합니다.

===

앞으로도 저희 돌봄센터와 사용하시는 분들을 위해서 꾸준하게 업그레이드를 하겠습니다.
불편사항이 있으시다면, 언제든지 말씀해 주시면 감사하겠습니다.

"Stamp-Management-System"은 Python, Streamlit, Supabase를 통해서 구현했습니다.
다시 한번 감사의 말씀 전해드립니다.
